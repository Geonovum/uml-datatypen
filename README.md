# uml-datatypen

In dit repository worden drie packages beheer atomaire datatypen die we binnen Geonovum gebruiken bij het modelleren met als uiteindelijk doel dat alle de Geonovum beheerde modellen deze packages gaan gebruiken. 

De volgende onderdelen worden beheerd:

| map      | omschrijving                                                                                                            |
| -------- | ----------------------------------------------------------------------------------------------------------------------- |
| xmi      | bevat een kopie van de xmi bestanden die je in UML kunt gebruiken. Beter is het de in svn beheerde versie te gebruiken. |
| Imvertor | bevat mapping bestanden die Imvertor kan gebruiken                                                                      |
|          |                                                                                                                         |

## XMI

Deze bevat drie collecties datatypen in drie bestanden

| bestand                     | omschrijving                                                                                |
| --------------------------- | ------------------------------------------------------------------------------------------- |
| datatypen-MIM.xml           | Bevat alle datatypes zoals beschreven in het (MIM)[https://docs.geostandaarden.nl/mim/mim/] |
| datatypen-ISO19107-2003.xml | Bevat ruimtelijke datatypes zoals beschreven in deze standaard                         |
| datatypen-overig.xml        | Bevat alle datatypes die in Geonovum informatiemodellen gebruikt worden en die niet in de andere catagorieÃ«n vallen |
